# Babel presets, plugins and environment configurations

1. Presets: `env, flow`
2. Plugins: `emotion`

```sh
yarn add babel-preset-mkitt -DE
```

Add to package.json:

```json
"babel": {
  "presets": [
    "mkitt"
  ]
}
```
